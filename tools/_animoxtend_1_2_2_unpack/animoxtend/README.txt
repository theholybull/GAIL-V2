【简介】

AnimoXtend 是一款面向 3D 动画与创意制作的 Blender 插件，不仅提供高效的动画生成和重定向工具，还专注于智能化与自动化工作流。插件支持批量处理和自动优化，极大地减少繁琐操作，提升创作效率。AnimoXtend 不只是工具的集合，更是为动画师和创作者打造的创新平台，以提升创意表达和技术效率为核心，真正实现从创作到优化的全流程智能支持。

【安装方法】

1. 启动blender，在顶部菜单点击 编辑(Edit) → 首选项(Preference) → 插件(Add-ons) → 安装(Install) ，选择插件zip文件进行安装。

2. 在插件菜单里启用“AnimoXtend”插件。

3. 在插件列表中找到“AnimoXtend”，点击 “>”，展开 AnimoXtend 插件的信息，在 偏好设置(Preferences) 下方的 API Key 输入框中，填入 API Key。

【OVERVIEW】

AnimoXtend is a Blender add-on designed for 3D animation and creative production. It not only offers efficient tools for motion and model generation but also focuses on intelligent and automated workflows. The add-on supports batch processing and automatic optimization, significantly reducing repetitive tasks and enhancing creative efficiency. AnimoXtend is more than just a collection of tools; it is an innovative platform designed for animators and creators, with a core focus on enhancing creative expression and technical efficiency, providing full-process intelligent support from creation to optimization.

【INSTALLATION】

1. Start blender. In the top menu, click Edit → Preferences → Add-ons → Install, then select the add-on ZIP file to install it.

2. In the Add-ons menu, enable the "AnimoXtend" add-on.

3. In the add-ons list, locate "AnimoXtend" and click ">" to expand the plugin information. Under the "Preferences" section, find the "API Key" input box and enter your API Key.
